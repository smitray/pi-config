## Indian PAN Card Format and Verification

### PAN Structure: AAAAA0000A (10 characters)

Position 1-3: Random alphabets (series AAA to ZZZ)
Position 4: Assessee type
Position 5: First letter of surname/last name
Position 6-9: Sequential number (0001 to 9999)
Position 10: Alphabetic check digit

### Assessee Types (4th Character)

| Code | Type |
|------|------|
| A | Association of Persons (AOP) |
| B | Body of Individuals (BOI) |
| C | Company |
| F | Firm |
| G | Government |
| H | HUF (Hindu Undivided Family) |
| L | Local Authority |
| J | Artificial Juridical Person |
| P | Individual (Person) |
| T | Trust (AGRI) |

### Example: FSIPR0296G

- FSI: Random characters
- P: Individual (Person)
- R: Surname starts with 'R'
- 0296: Sequential number
- G: Check digit

### How to Verify PAN (Authorized Methods)

1. Income Tax e-Filing Portal (Recommended)
   URL: https://www.incometax.gov.in
   Steps: Register → Login → Dashboard shows all details

2. Quick PAN Verify (No Login)
   URL: https://www.incometax.gov.in/iec/foportal
   Steps: Services → View PAN Details → Enter PAN → CAPTCHA

3. NSDL Instant Verify
   URL: https://tin.tin-nsdl.com/pan/verify.html
   Steps: Enter PAN → CAPTCHA → Instant result

### What's Protected (Private)

- Full name
- Date of birth
- Address
- Father's name
- Jurisdiction
- Status (active/inactive)

These are private and stored in Income Tax Department's database. Cannot be accessed publicly for privacy/security reasons.

### Legitimate Verification Use Cases

| Use Case | Method |
|----------|--------|
| Own PAN verification | Income Tax portal (register and login) |
| KYC verification | Authorized verification process |
| Employment background check | Authorized agencies (First Advantage, AuthBridge) |
| Financial due diligence | SEBI/ROC public records for companies |
| Legal proceedings | Court can order disclosure through proper channels
