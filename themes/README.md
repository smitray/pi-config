# `themes/`

User-level pi themes. Each `*.json` file is one theme, loaded by name from
`settings.json` (`"theme": "..."`).

| Theme | Notes |
|-------|-------|
| [`catppuccin-mocha.json`](./catppuccin-mocha.json) | Catppuccin Mocha palette |

To add a theme: drop a `*.json` file here, set `theme` in `settings.json` to
its basename, and restart pi.
