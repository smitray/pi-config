# Knowledge Base

This `.kb/` directory is a Karpathy-style LLM Wiki maintained by the `kb` pi extension.

## Skills (auto-loaded when KB extension is active)

- `kb` — full tool reference
- `kb-research` — research & save to KB
- `kb-capture-url` — capture GitHub/docs/YouTube
- `kb-bootstrap` — init a new vault
- `kb-update` — update existing pages

## Layout

```
.kb/
├── raw/sources/    # immutable source packets
├── wiki/           # agent-owned pages (concepts/entities/syntheses/analyses/...)
├── meta/           # auto-generated registry, backlinks, embeddings, events
└── templates/      # page templates
```
